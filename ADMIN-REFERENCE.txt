ADD PRODUCT
───────────
1. Click "Products" tab → "+ Add Product"
2. Fill required fields (*):
   - Product Name *
   - Price *
   - Category * (dropdown)
   - Description (optional)
   - Image (optional, max 5MB)
3. Add Variants (optional):
   - Size: XS, S, M, L, XL, 2XL, 3XL, One Size
   - Color: 12 color options
   - Stock: number of units
4. Click "Create Product"
5. Product appears on storefront instantly

EDIT PRODUCT
────────────
1. Go to dashboard
2. Find product in products list
3. Click "Edit" 
4. Modify any field
5. Update variants (add, remove, or change stock)
6. Click "Update Product"
7. Changes live immediately

DELETE PRODUCT
──────────────
1. Go to products list
2. Click product
3. Look for "Delete" button
4. Confirm deletion
5. Product removed from storefront & database

MANAGE INVENTORY
────────────────
1. Go to Products
2. Click product name
3. View/edit variants section
4. Adjust stock counts
5. Add new size/color combinations
6. Remove variants no longer needed
7. Save

ORDERS
──────
View all orders:
1. Go to dashboard
2. Click "Recent Orders" or "Orders" tab
3. See all orders sorted by date (newest first)

Process an order:
1. Click order number or "View"
2. See customer details & items ordered
3. Update status:
   - Pending → Completed (when payment received)
   - Completed → Shipped (when sent)
   - Any → Cancelled (if needed)
4. Status automatically updated in system

Troubleshooting
───────────────

Forgot password?
  → Contact server admin (SSH into Empire)
  → Reset via MySQL

Product not showing on storefront?
  → Check if product has image
  → Verify price is set
  → Check product is in database
  → Try clearing browser cache

Orders not appearing?
  → Check if order status updated
  → View in MySQL: SELECT * FROM orders;
  → Check Stripe dashboard for failed payments

Stripe issues?
  → Verify test vs live keys in config.php
  → Check Stripe dashboard for errors
  → Review webhook logs
